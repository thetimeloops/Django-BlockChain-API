# Django BlockChain API

Contains BlockChain Endpoints.

## Features
* Create Block
* Get Last Block
* Proof Of Work
* Chain Validity
* Transaction Add
* Node Add
* Chain Replacement
* Block Mining
* Get Chain
* Replace Chain

## Usage

```bash
cd django_blockchain_api
python3 manage.py runserver
```
